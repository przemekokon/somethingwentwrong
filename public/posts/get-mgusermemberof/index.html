Which groups do users belong to? A Graph API approach

March 23, 2026 · 1 min · 73 words · Przemek |  Suggest Changes

What’s that?#

Just another script for listing users and their group memberships. You feed it a text file with UPNs, it queries Microsoft Graph, and spits out a flat CSV like:

UserPrincipalNamePrimarySmtpAddressGroupNameGroupEmailGroupType
user1@abc.comuser1@abc.comSome DLsome-dl@abc.comDistributionList
user1@abc.comuser1@abc.comSecurity Group ABCsg-abc@abc.comMailSecurityGroup
user2@abc.comuser2@abc.comSome DLsome-dl@abc.comDistributionList

Where’s it?#

On GitHub

Requirements#

  • PowerShell 5.1+ or PowerShell 7+
  • Microsoft Graph PowerShell SDK
  • Permissions: User.Read.All, GroupMember.Read.All, Group.Read.All
  • Powershell
  • Graph-Api
  • EntraID
  • Exchange
